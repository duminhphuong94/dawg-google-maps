Google Maps JS Basic: Let's Start - Creating A Map

Creating a Google Map with Javascript is easy, merely having some mandatory details or fields that need to be in place. We'll first need (for this example), a Latitude and Longitude Coordinates which is invoked with new google.maps.LatLng(Lat, Lng), a specific Zoom Value and a center Map Center that is based on the Lat-Lng Coordinates.

By default, the mapTypeId is roadmap. Let's have a look at the other map types below.

Google Maps - Rotating Imagery

Satellite & Hybrid mapy types provides an additional feature for rotating imagery, starting at certain higher zoom levels and if available for that particular location. When a user zooms into a particular location, (provided this service is available for that location), the map image changes to a 45 degrees tilt.

The map is also provided with the Rotation Controls.
