Google Maps - Rotating Imagery

Satellite & Hybrid mapy types provides an additional feature for rotating imagery, starting at certain higher zoom levels and if available for that particular location. When a user zooms into a particular location, (provided this service is available for that location), the map image changes to a 45 degrees tilt.

The map is also provided with the Rotation Controls.

In the below example, the location being shown is Central Park, New York City, a JS setInterval of every 5 seconds is set to make the image rotate.

Google Maps - Creating Custom Base Map Types

Creating a custom base map type invloves in understanding certain mandatory requirements. This custom base map type once created, would generate as Grids on to the main parent map holding element, initialzed with the help of creating a Class [Constructor Function] in Javascript. Necessary style attributes can be give to each of these grid as required.

Once created, you can console.log the arguments from the getTile() method from the Constructor's prototype object. A sample snippet of what you could get would be:
