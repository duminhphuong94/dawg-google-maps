Google Maps Javascript API - Geocoding Service

Please ensure to have Google Maps JavaScript API and the Google Maps Geocoding Service API enabled at https://console.developers.google.com/apis/dashboard post creating your API key

Overview

Google Maps Javascript API - Geocoding Services refers to converting one form of location detail to another form, which could be either a Human Readable address value or a Latitude-Longitude Coordinate into a Human readable value. In order to be able to use this service, a geocoding object would need to be first created, followed by supplying this object with various options. The new object is created with new google.maps.Geocoder(), to which the .geocode(optionsObject, callbackFunc) method is called upon.

The examples below have various forms of results which are displayed on the respective Google Map Element. You can click on the bouncing markers to get an idea of what kind of results are being displayed. You can also refer to the example code attach to each.

To get a better in-depth understanding of how this works, please visit Google Maps Javasript API - Geocoding Service for more information.
