Gesture Handling - Cooperative

This gestureHandling mode is the default mode for Google Maps, indicating that google maps centers nor bounds change when scrolling over them. Applicable either on mobile or desktops. Changes can be made only upon actually Dragging the map (on desktop) or using a Two Finger Movements for mobile and other similar devices.

Gesture Handling - None

This mode indicates that Drag Feature on desktops and Pinching & Panning on mobile and similar devices are disabled. In this example, while gestureHandling has been set to none, the map's Zoom Feature has also been disabled.

Gesture Handling - Greedy
