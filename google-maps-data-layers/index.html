Google Maps Displaying Data - Load GeoJson and Styling

Google Maps Data Layer's loadGeoJson() method helps create a map based representation of any file which contains arbitrary GeoJson Data. The GeoJson format is currently a standard format for sharing Geospatial Data over the internet. Styling for these geospatial representaion over Google Maps can be done with the help of the setStyle() method. While you may be able to apply different style options all together, they will only bring about an effect for the required element on the map.

Our example here, brings in data from a GeoJson Endpoint, https://phl.carto.com/api/v2/sql?q=SELECT+*+FROM+li_demolitions&filename=li_demolitions&format=geojson&skipfields=car, post its representation on the map, each marker is changed from the default Google Maps Marker to our customized marker icon.

Google Maps Data Layer Events

Continuing from our previous example, we are representing GeoJson Data about Building Demolitions that have happened in the city of Philadelphia. Google Map's map.data object can be bound with events with .addListener('event_name', callback) method. Here we have bound a click event to the map.data object, which is responsible for triggering the respective Info Window and providing a Bounce Animation to the marker on the map.

Additionaly, the Map itself is also bound with a click event which removes any Info Window associations with it and also helps remove any map.data.overrideStyle() method details. Similarly, the Info Windows also have events that have been bounds to event, with a closeclick event to dissassociate itself from the map and also remove styling options rendered by map.data.overrideStyle() method.

The map.data.revertStyle(); method is responsible for removing any styling options that have been set by the map.data.overrideStyle() method.
