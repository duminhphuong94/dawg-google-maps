Google Maps Javascript API - Drawing Library

Imagine you're a Military General, charting out your next plan to attack your enemy! Drawing on a map has never been so much fun. Yay!

Overview

Google Maps Javascript API - Drawing Manager, allows us to create overlays instantly over a map or geographical regions. These overlays are similar to the Google Maps Javascript API - Shapes like Circles and Rectangles. To be able to use this feature, an initial setup needs to be done to have this library imported into your work. Please also ensure to enable the Google Maps Javascript API in your respective dashboard once your API key has been generated.

<script type="text/javascript" src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&libraries=drawing"></script>

Checkout the examples below and refer to the example codes attached to each of them to get started. To get a better in-depth understanding of how this works, please visit Google Maps Javasript API - Drawing Layer(Library) for more information.

Google Maps Drawing Manager - Create Drawing Manager

Once you've setup your workspace for this API, the first step is to create a class instance/object of the drawing manager with new google.maps.drawing.DrawingManager(). Within this the drawing manager options object is placed that helps determine what options would be available when a user interacts with the map.

To highlight some of the options used in the example below, the drawingMode option helps determine the initial Drawing Option that woudl be active as default when clicking and drawing on your map. This helps also highlight a dark outeredge to the option on the Controls attached on to your map. Setting this value as null means that the user would have to select an option from the control and only post this the drawing manager actually become active.

The drawingControl field receives a Boolean value, indicating whether it would be present on the map or not. The drawingOptions field is an object, which can help set the position of the controls on your map and also determine what shapes or overlays would be available to the user.

In the example above, while the drawing tools/controls have been enabled, you would not be able to draw any overlays as the drawingMode field is set to null. Select an option to draw and interact with the map.

Google Maps Drawing Manager - Events Drawing Manager

Whenever a Drawing Manager is used to create an overlay, there are two kinds of events which are fired. The first type is the Type of Overlay Completed event and the other is the general Overlay Completed Event. The Type of Overlay Completed event is specific to that of a Circle or a Rectangle whereas the general Overlay Completed Event signifies some overlay has been completed and based on the kind of overlay created, you would then have access to all methods specific to that overlay.

The methods for each of the overlay are the same as the methods for the Google Maps Javascript API - Shapes. To start off with the events for this particular class, open up your browser console and create an overlay to see the event.type and event.overlay results. You could refer to Drawing Events and Overlay Completed Event Object Specifications for more information.
