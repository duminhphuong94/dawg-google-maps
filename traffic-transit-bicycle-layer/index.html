Google Maps Javascript API - Traffic, Transit and Bicycle Layers

Please ensure to have Google Maps JavaScript API enabled at https://console.developers.google.com/apis/dashboard post creating your API key

Overview

Google Maps Javascript API Traffic, Transit and Bicycle Layers are overlays that provide users information about current Traffic Conditions, Transit and Bicycle routes. Please note that information available on these overlays may be available only in select regions.

Please refer to Google Maps Javascript API - Traffic, Transit & Bicycle Layer for more information.
