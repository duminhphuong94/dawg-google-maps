Google Maps Distance Service - Basics

We'll start off with the basics on how to acquire Distance Information from one Origin to one destination using this particular service. To create a Distance Service Request, we'll need to have an options object with carries with it some mandatory fields, namely origins, destinations and travelMode. Origins and Destinations here in this service request are of data type Array. Then, create a new google.maps.DistanceMaxtrixService() object, upon which we'll call the getDistanceMatrix() method passed with 2 arguments, the intial options Object and a callback function to handle the response from this API call.

In our example below, our origin address is the Red Square and the destination address is the Moscow Zoo. An Info Window is generated merely to show the final result of the API call.

To get a better in-depth understanding of how this works, please visit Google Maps Javasript API - Distance Matrix for more information.

Google Maps Distance Service - Transit Travel Mode and Transit Options

Similar to Google Maps Directions Serice, the Distance Matrix Service has a Travel Mode option field, which when passed with a value of "TRANSIT", would provide you access to the Transit Options Object. This Transit Option Object can have fields like departureTime and modes. In our example below, the distance service request from the Red Square to the Moscow Zoo, provides information with a TRANSIT travel mode, additionally also with the Transit Options Object.

The results for this particular example is shown after the map.

To get a better in-depth understanding of how this works, please visit Google Maps Javasript API - Distance Matrix and search for Travel Modes and Transit Options.

Google Maps Distance Service - Unit Sytems

The Unit System option has two possible values, which are google.maps.UnitSystem.IMPERIAL or google.maps.UnitSystem.METRIC. This particular field can be added to the Distance Service Object, which post acquring the API call result, would determine the way unit systems are displayed as information for your map.

In our example below, the Distance Service request result from the Red Square to the Moscow Zoo now has its distance given in Miles(shown in the overlay InfoWindow), as we have placed the value of google.maps.UnitSystem.IMPERIAL for its unit system field.

To get a better in-depth understanding of how this works, please visit Google Maps Javasript API - Distance Matrix and search for Unit System.
