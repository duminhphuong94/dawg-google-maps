Google Maps Directions Service - Basics

Creating a basic Direction Service Request requires certain options in the Directions Service Object to be present on mandate. These include Origin, Destination and Travel Mode. Initate a request by creating a service request object with new google.maps.DirectionsService() followed by a directions renderer object with new google.maps.DirectionsRenderer(), whose setMap() method to carry the map onto which the request results needs to be displayed on.

You would the need to invoke the route() method of the Direction Service Object, passing it the Directions Options Object, which contains the mandatory fields, with a callback function to process the required result.

In our example below, a Direction Service Request is placed for getting information between New York City and Philadelphia with the Travel Mode as Driving. To get a better understanding of this, visit the example code details for this section. Also, view the results of the API call made with your browser's Console Logs.

Google Maps Directions Service - Waypoints

While acquiring routes from one particular location to another can generate a path with just mandatory fields (like the 1st Example), Google Maps Routes can also be generated by providing specific way points that a particular route has to take to reach a final destination. Providing waypoints may have an effect on the route being generated, meaning it could be different from a route that has no specific way points attached. This is done by passing the Waypoints field into the directionOptions object, within which, Location and Stopover properties needs to be given.

If a particular waypoint on route to the final destination is set as stopover: true, this splits the entire route into two parts (Point A -> Waypoint and then Waypoint -> Final Destination). Depending upon the number of stopovers set as true, Google Maps API would split one route into sub-routes.

To get a better and in-depth understanding of the various options that would be available, please refer to Google Maps Javascript API - Directions and search for the section that has details of Using Waypoints in Routes.

Our example above, set over New York, has a Direction Service Request from Brooklyn Bridge to Robert F Kennedy Bridge, but passing the Waypoint Field into the service request object, with one waypoint set to stopover: true, splits the full route into two parts.
