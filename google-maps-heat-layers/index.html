Google Maps Displaying Data - Heat Layer

The heatmap visualization of data on your Google Map, provides visual indication of how intense some data can be. Post being provided some intial inputs, the Heat Layers gets generated as an overlay, appearing over the centered location of the map. Apart from ensuring that the standard Javascript API Library is loaded, you'll also need to ensure that the Visualization library is passed as an additional parameter to the API url.

<script type="text/javascript" src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&libraries=visualization"> </script>

A Heatmap Layer is generated by creating a new heatmap object, new google.maps.visualization.HeatmapLayer() and passing an Array of new google.maps.LatLng(lat, lng) objects into it or using a Weighted Location Object.

The example above showcases how GeoJSON data can be used to generate a Heatmap Layer. Here, an API call is made to an endpoint, which returns the result of a successful call. Once you've located the parts of the results that contain Geospatial information (Lat-Lng Coordinates), place them in an Array and pass that array as the value for the data field in the Heatmap's Options Object.
