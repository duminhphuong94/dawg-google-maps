Google Street View Image API

Learn about the Google Street View Image API, which allows you to embed a static non-interactive Street View Image on your webpage. Please ensure to enabled the Google Street View Image API at https://console.developers.google.com/apis post creating your API key

Overview

Embed is done with an <img> tag on your webpage, which carries an src attribute values that finally helps display the map. The image request URL is https://maps.googleapis.com/maps/api/streetview?parameters.

Refer to Google Street View Image API Guide for further information.

Street View: Basics - I

Using the <img> html tag, you can generate an image with its https://maps.googleapis.com/maps/api/streetview?key=YOUR_API_KEY&location=27.1750,78.0422&size=1920x450&heading=360, which shows the Taj Mahal. Here, the location parameter has been provided with Latitude-Longitude coordinates along with the size of the image and the heading parameter.

Street View: Basics - II

Using the <img> html tag, you can generate an image with its https://maps.googleapis.com/maps/api/streetview?key=YOUR_API_KEY&location=grand+canyon+usa&size=1920x450&heading=295, which shows the Grand Canyon. Difference from the previous image being that instead of providing Latitude-Longitude coordinates, the location parameter is provided with text appended with +.

Street View: FOV and Pitch

The fov [a.k.a Field Of View], determines the horizontal view of the location. Smaller values represents zoom in(s) with a maximum allowed of 120. The pitch parameter represents the up or down angle of the camera relative to that particular location picture. For example, to generate the image of the Pyramids like shown, use https://maps.googleapis.com/maps/api/streetview?key=YOUR_API_KEY&location=29.9773,31.1325&size=1920x450&fov=50&pitch=20.

Street View: Imagery or Location Not Found?

Sometimes there is a possibility that you would not get an image from the Google Street View Image API. Go ahead and give a request for something like https://maps.googleapis.com/maps/api/streetview?key=YOUR_API_KEY&location=planet+pluto+milky+way+galaxy&size=1920x450.
