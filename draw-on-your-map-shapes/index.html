The example here would help generate a Polyline Path on your Google Map when a user clicks on it. Please note that the intial path is created only after two points on the map have been clicked, although every click (including the first one) is being registered.

Google Map Shapes - Inspecting A Polygon

In this example, we will be examing the various Latitude-Longitude Coordinates that have been used to setup the Polygon Shape. To inspect/retrieve the polygon Lat-Lng coordinates with which it was built, a Click Event has been attached to the polygon area here. Then the getPath() method is used to retrieve all the paths in the polygon array. You could at this stage iterate through using a for-loop to get each single point using the getAt() method.

Interact with this example, you'll notice that the click event not only specifies the specific Latitude-Longitude coordinates for the click but also provides 5 Coordinate Points which are responsible for creating this polygon shape.

Google Map Shapes - Hole(s) in a Polygon

Any space inside a Polygon Area can be created, which would create a Hole effect. This exaple demonstrates just that. However, the only issue with using the regular Polygon Class in Google Maps is that these inner Coordinates that creates holes, needs to be placed in an order opposite to the placement order of the outermost Latitude-Longitude Coordinates.

The paths field in the Polygon Options Object, would take in an Array containing Arrays where each would carry the list of coordinates. The outermost area that needs to be covered would be placed first in the list of Path Arrays.

Rectangle

This section provides the basic understanding of how to implement simple Rectangles on your google map. Please ensure to have Google Maps JavaScript API enabled at https://console.developers.google.com/apis/dashboard post creating your API key

Key Point: Unlike polygon shapes, Rectangles require a bounds property to be defined in its options object, carrying the respective North, South, East & West Lat-Lng Coordinates. These coordinates are generated from a Google new google.maps.LatLngBounds Class from which you could incorporate its toJSON() method to retrieve bounds info.

Google Map Shapes - Removing Rectangle

A Rectangle similar to that of Polygon shapes, can be disassociated/removed from a map by setting its setMap() argument value to null. To place the rectangle on the map again, pass the respective map variable as an argument again.

Google Map Shapes - Removing Circle

A Circle Shape can be disassociated/removed from a particular google map, by setting its respective setMap() method argument to null. It can be placed again by providing the respective map variable as the argument again.
