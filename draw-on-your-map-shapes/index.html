Google Map Shapes - Simple Polyline

A Polyline is constructed as an overlay on your Google Maps. This is done with the help of an array of Latitude-Longitude Coordinates along with some other fields that together comprises the Polyline Options Object.

In our example here, the map starts off with its center at the country of Brazil. The red colored Polyline Path starts off at Sao Paolo, moving south-wards to Asuncion, futher south to Cordoba, then heading north towards Quito and then finally connecting back at Sao Paolo.

Google Map Shapes - Remove Polyline

The simplest way to remove a polyline from a particular map, is to have its map field in its Polyline Options Object set to null. This example depicts the same path with a polyline as the previous one. The difference being that every 2 seconds, the polyline is removed from the map and then placed in again.

Google Map Shapes - Complex Polyline [Dynamic Example]

The example here would help generate a Polyline Path on your Google Map when a user clicks on it. Please note that the intial path is created only after two points on the map have been clicked, although every click (including the first one) is being registered.

You can interact with this map by first clicking at any point on the map, followed by clicking on another location. By continuing this process, it will help create a trailing Polyline Path on all the places that you have clicked.

Google Map Shapes - Basic Polygon

Polygons in Google Maps are generated with the help of the Polygon Object Options. The object in this example, would take in the neccessary Latitude-Longitude Coordinates which is in the form of an array, followed by strokeWidth, strokeColor, strokeWeight, strokeOpacity, fillColor, fillOpacity. Don't forget to attach the Map to which the polygon needs to be generated upon.
