Map Markers

Everything that you'll need to get upto speed with creating and placing Markers on your google maps. Just remember that markers are nothing but a kind of overlays.

Overview

Markers on google maps help identify a particular location, by default being created with a specific image by googgle maps. These markers can also be customized with our own images of Icons that we can place on our maps. Markers can also be interactive, for example, clicking on a marker could help pop-up an Info Window.

Please ensure to have Google Maps JavaScript API enabled at https://console.developers.google.com/apis/dashboard post creating your API key

Refer to Google Places API Markers for more information.

Google Maps Markers: Let's Start - Basics

A basic marker placement is relatively easy once you've built your required google map. In order to place a marker on your map, you'll need a specific Lat-Lng Coordinate and the Map onto which you'll place the marker on. Markers get created with the new google.maps.Marker() constructor which takes in an object with the Coordinates and Map settings.

In our example here, the location is Hawaii with the marker being placed onto the Lat-Lng Coordinates that was used to create the center for this map. Use your mouse and hover over the marker and a Title Box with a small message would appear.

Google Maps Markers: Remove Markers and Animation

Markers on a set Lat-Lng point on your google map can be removed by using the marker.setMap(null) method. This however, does not delete the marker itself. If you do have multiple markers to manage, you could place them in an Array and empty the array should you wish to disassociate all markers from your map.

Animation options are also availabe for markers, available in two options provided by google maps api - google.maps.Animation.BOUNCE & google.maps.Animation.DROP. This example demonstrates markers being set on the map with an animation and then remove from the map, only to be placed at a different location.
