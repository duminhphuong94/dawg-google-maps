Google Map Controls - Disable Default UI

All maps created with the Javascript API, will be rendered with certain default controls. If you would like to completely disable all controls from your map, place disableDefaultUI to true in your Map Options Object. This example displays a google map without any controls being displayed on it.

Google Map Controls - Map Type Controls

This particular option set within the Map Options Object comes with three possible values: google.maps.MapTypeControlStyle.HORIZONTAL_BAR, google.maps.MapTypeControlStyle.DROPDOWN_MENU and google.maps.MapTypeControlStyle.DEFAULT.

Positioning Map Controls

All google map controls contains a certain position property that would enable you to position these controls at certain parts on your map. While the values pertaining to these positions are not Absolute in nature, they do come with certain pre-defined values. Compare the example below with a regular map being set without making changes to its controls.

You would notice that the positions of our map's controls have been changed from their reguar/default position. Depending upon what controls are available at what times (for eg, rotation controls being available at a specific zoom point), making changes in the Map Options Object would have its effects on these controls.

Please refer to Google Maps API - Control Positioning for more information.

Google Map Controls - Creating A Custom Control

While we haven't had a look at attaching events to our google maps, the example here showcases the creation of a custom UI. This particular <div> element along with its hover effects and font styles are initially placed in an external css file with a particular class name. This is then set as an attribute when dynamically creating a <div> element with Javascript.

Once your custom element is generated as you wanted it, you could then attach an event to it (here we're attaching a Javascript onclick event to reset the ceter of the map to its initial coordinates).

Google Map Controls - Storing a State with Custom Controls

This example demonstrates how we could store a certain value on a button click event attachment and then, retrieve those same values with the click of another button. Here, move the map to a different location and click on the Set This Center button. This will now set your current map position as a default position.

Move around the map again and then click the Reset Position button, which will then move back the map to the same point where you had initially stored it as the default center location.
