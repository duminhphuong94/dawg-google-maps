DAWG Google Maps

A self learning project created to help you get started on understanding the fundamentals of Google Maps API (Client Side). Start off with really simple stuff to some advanced topics.

Start with Simple Implementations

Get yourself started by going through the very basics of Google Maps Javascript API. These topics include Google Embed Maps API, Google Static Maps API and Google Street View Image API. Plus also visit the Google Maps JS API section on this website to get information on how to use basic Javascript (with some help from JQuery as well) to create maps.

Open Up Example Codes

Every section that is covered in this website, comes along with an example code. The example code is placed as an image that pops up. Another way to understand how things work is with your Browser Console. Certain API responses have been logged in to give you an idea of what you're dealing with.

Ride some Advanced Topics

This website aims at providing the maximum possible detail on a variety of Advanced Topics like Google Maps Geometry Library and Google Maps Autcomplete and Search Box Feature. Once you've got the hang of these, you should be able to even build you own basic application and launch it on the web.
