Google Maps Javascript API - Elevation Service

Please ensure to have Google Maps JavaScript API and the Google Maps Elevation API enabled at https://console.developers.google.com/apis/dashboard post creating your API key

To get a better in-depth understanding of how this works, please visit Google Maps Javasript API - Elevation Service for more information.

Overview

The Google Maps Javascript API - Elevation Service helps provide information about the Elevation of a particular location or a group of location from their respective Lattitude and Longitude Coordinates. The initial step is to create an Elevation Service Object with new google.maps.ElevationSerivce(). You would also need an Array of Lat-Lng Coordinates which would then be used as reference points upon which the API call needs to be made.

Our first two examples demonstrate merely acquring elevation values for the locations. This is done using the getElevationForLocations() method to which the Lat-Lng Array and a callback function to handle the final API response, are passed as arguments.

The last example is about acquiring Elevation Details along a path from an origin point to a final point along the path line. Here the getElevationAlongPath() method is used, an object containing the Path Array and Sample Numbers along with a callback function to handle the API response, are passed as arguments.
