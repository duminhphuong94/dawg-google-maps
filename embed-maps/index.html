Google Maps Embed API

An embed version for placing google maps onto your web-pages without having to code in Javascript. Please ensure to enable the Google Maps Embed API at https://console.developers.google.com/apis/dashboard post creating your API key.

Overview

Embed is done with an iframe on your webpage, which carries an src attribute values that finally helps display the map. There are 5 types of modes:

  • Place Mode [ place ]
  • Directions Mode [ directions ]
  • Search Mode [ search ]
  • View Mode [ view ]
  • StreetView Mode [ streetview ]

An example URL that is required for this kind of API request is https://www.google.com/maps/embed/v1/MODE?key=YOUR_API_KEY&parameters. Refer to Google Maps Embed API Guide for further information.

Embed Maps: Place Mode

Embed a map location onto your webpage with place mode in the iframe url. For example to display the map onto the right, the src value for the iframe would be https://www.google.com/maps/embed/v1/place?key=YOUR_API_KEY&q=Central+Park+New+York.

Embed Maps: Direction Mode

Embed a map location onto your webpage with directions mode in the iframe url. For example to display the map onto the right, the src value for the iframe would be https://www.google.com/maps/embed/v1/directions?key=YOUR_API_KEY&origin=trump+tower+newyork&destination=central+park+newyork.

Embed Maps: Search Mode

Embed a map location onto your webpage with search mode in the iframe url. For example to display the map onto the right, the src value for the iframe would be https://www.google.com/maps/embed/v1/search?key=YOUR_API_KEY&q=indian+restaurants+in+tokyo.

Embed Maps: View Mode

Embed a map location onto your webpage with view mode in the iframe url. For example to display the map onto the right, the src value for the iframe would be https://www.google.com/maps/embed/v1/search?key=YOUR_API_KEY&center=36.1699,-115.1398&zoom=15&maptype=satellite.

Embed Maps: StreetView Mode

Embed a map location onto your webpage with streetview mode in the iframe url. For example to display the map onto the right, the src value for the iframe would be https://www.google.com/maps/embed/v1/streetview?key=YOUR_API_KEY&location=29.9773,31.1325&heading=210&pitch=30&fov=35.
